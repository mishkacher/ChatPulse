# ChatPulse

**ChatPulse** — минимальная нативная утилита для строки меню macOS, которая следит за выбранными чатами ChatGPT в Google Chrome и отправляет команду:

> продолжай и не останавливайся до технического лимита

Приложение не использует платные API, облачные модели или отдельный ИИ-агент. Оно управляет только указанными чатами через Google Chrome и не пытается обходить технические лимиты ChatGPT.

## Что уже реализовано

- нативное приложение для строки меню macOS;
- одна переключаемая кнопка **«Запустить / Остановить»**;
- статус текущей работы;
- ручная проверка через **«Проверить сейчас»**;
- интервалы 1, 2, 5, 10, 15 и 30 минут;
- произвольный интервал от 30 секунд;
- добавление текущего чата из Google Chrome;
- сохранение названия, URL и состояния каждого чата;
- отдельное включение и отключение каждого чата;
- открытие сохранённого чата из меню;
- защита от повторной отправки одной и той же команды;
- ожидание полного ответа ассистента;
- технический лимит и ошибки страницы блокируют отправку;
- журнал последних действий;
- подтверждение, что сообщение действительно появилось в чате;
- 20 автоматизированных quality gates и unit-тесты.

## Главная логика

ChatPulse не отправляет команду сразу после обнаружения нового ответа.

Пример при интервале 5 минут:

1. На первой проверке приложение видит новый ответ и запоминает его.
2. Через 5 минут оно проверяет этот чат снова.
3. Если ответ всё ещё тот же, генерация завершена и технического лимита нет, приложение отправляет команду продолжения.
4. Когда ChatGPT отвечает, отпечаток последнего сообщения меняется.
5. На следующей проверке ChatPulse только фиксирует новый ответ и ничего не отправляет.
6. Команда будет отправлена лишь на ещё одной проверке, если ответ остаётся завершённым и неизменным.

Таким образом, уже ответивший чат не перезапускается сразу на ближайшей проверке.

## Требования

- macOS 13 Ventura или новее;
- Google Chrome;
- Swift 6 / Xcode Command Line Tools для сборки из исходников;
- авторизованный аккаунт ChatGPT в Chrome.

## Установка из исходников

```bash
git clone https://github.com/mishkacher/ChatPulse.git
cd ChatPulse
bash scripts/install_app.sh
```

После сборки приложение будет установлено в:

```text
/Applications/ChatPulse.app
```

## Первичная настройка Chrome

Chrome запрещает выполнение JavaScript через Apple Events по умолчанию. Это необходимо включить один раз:

1. Откройте Google Chrome.
2. В верхнем меню выберите **View → Developer**.
3. Включите **Allow JavaScript from Apple Events**.
4. Запустите ChatPulse.
5. При первом обращении macOS попросит разрешить ChatPulse управлять Google Chrome — подтвердите.

Разрешение можно проверить в **System Settings → Privacy & Security → Automation**.

## Использование

1. Откройте нужный чат ChatGPT в Chrome.
2. Нажмите значок ChatPulse в строке меню.
3. Выберите **«Добавить текущий чат Chrome»**.
4. Повторите это для остальных чатов.
5. Установите интервал проверки.
6. Нажмите **«Запустить»**.

ChatPulse сохранит список после перезапуска приложения и Mac.

## Хранение настроек

Настройки хранятся локально:

```text
~/Library/Application Support/ChatPulse/settings.json
```

В файл записываются только:

- названия и URL выбранных чатов;
- включённое состояние;
- интервал проверки;
- отпечатки последних сообщений;
- время последней успешной отправки.

Пароли, cookies и содержимое всей переписки не сохраняются.

## Ограничения

- ChatPulse зависит от структуры веб-интерфейса ChatGPT. После крупных изменений интерфейса селекторы могут потребовать обновления.
- Приложение не обходит ограничения тарифа и не ускоряет сброс лимита.
- Chrome должен быть авторизован в ChatGPT.
- Для первой версии требуется включённый параметр **Allow JavaScript from Apple Events**.
- Сборка из репозитория подписывается локальной ad-hoc подписью, а не сертификатом Apple Developer ID.

## Безопасность

ChatPulse открывает и изменяет только сохранённые URL чатов на доменах `chatgpt.com` и `chat.openai.com`. Отправка выполняется только после подтверждения завершённого ответа и только фиксированным текстом из настроек приложения.

Перед использованием прочитайте [SECURITY.md](SECURITY.md).

## Архитектура

- `ChatPulseCore` — модели, состояние, дедупликация и хранение настроек;
- `ChatPulseApp` — интерфейс строки меню, таймер и управление Chrome;
- `ChromeAutomation` — AppleScript и JavaScript для чтения состояния чата и отправки команды;
- `DecisionEngine` — детерминированное решение, можно ли отправить продолжение.

Подробности: [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md).
