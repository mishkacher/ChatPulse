## Что делает ChatPulse

ChatPulse работает внутри уже авторизованного профиля Chrome и следит только за выбранными страницами `chatgpt.com`. Когда ответ ассистента завершён и остаётся неизменным полный интервал, расширение отправляет команду:

```text
продолжай и не останавливайся до технического лимита
```

Команду можно изменить в настройках.

## Возможности

- работа с обычным входом ChatGPT через Google в Chrome;
- несколько сохранённых чатов;
- интервалы от 30 секунд до 24 часов;
- последовательная проверка вкладок;
- безопасная первая проверка без отправки;
- защита `at-most-once` от повторной отправки одному ответу;
- автоматическое восстановление выгруженных, frozen и неотвечающих вкладок;
- плановое обновление безопасных неактивных чатов каждые 5–15 минут;
- защита активной вкладки, текущей генерации и пользовательского черновика;
- восстановление фоновой генерации, зависшей более 20 минут;
- повторный freshness-preflight непосредственно перед отправкой;
- добавление последнего использованного чата из полного интерфейса;
- открытие уже существующей вкладки без создания дубля;
- локальный журнал событий с причиной восстановления;
- скины **macOS** и **ChatPulse Preview**;
- версия интерфейса автоматически берётся из Manifest.

## Установка

1. Скачайте репозиторий или ZIP из GitHub Actions.
2. Распакуйте архив и сохраните папку в постоянном месте.
3. Откройте в Chrome адрес `chrome://extensions`.
4. Включите **Режим разработчика**.
5. Нажмите **Загрузить распакованное расширение**.
6. Выберите папку `chrome-extension`, внутри которой находится `manifest.json`.
7. Закрепите ChatPulse через меню расширений Chrome.

Подпись Apple, Safari Extension и отдельное macOS-приложение не требуются.

## Первый запуск

1. Войдите в ChatGPT в том же профиле Chrome.
2. Откройте конкретный чат с адресом вида `https://chatgpt.com/c/...`.
3. Нажмите значок ChatPulse.
4. Выберите **Добавить текущий чат**.
5. Установите интервал и нажмите **Запустить**.

В полном интерфейсе кнопка добавления выбирает последний использованный конкретный чат ChatGPT, даже когда сама страница настроек является активной вкладкой.

Первая проверка только фиксирует текущий ответ. Если ответ не изменился, следующая проверка может отправить команду ровно один раз.

## Как работает восстановление вкладок

Для управляемых вкладок ChatPulse пытается отключить автоматическую выгрузку Chrome. Перед чтением DOM расширение проверяет состояния `discarded` и `frozen`, доступность content script и наличие ошибки страницы. При необходимости вкладка безопасно обновляется, после чего ChatPulse ждёт загрузку и гидратацию интерфейса ChatGPT.

Неактивные вкладки без черновика и без текущей генерации периодически обновляются. Активную вкладку и заполненное поле ввода ChatPulse автоматически не перезагружает.

## Приватность и разрешения

Все настройки и служебные отпечатки хранятся локально в `chrome.storage.local`. Расширение не содержит телеметрии, серверной части, рекламы, внешнего ИИ и платных API.

Запрашиваются только разрешения:

- `alarms` — плановые проверки;
- `storage` — локальные настройки и журнал;
- `tabs` — поиск, открытие и восстановление сохранённых чатов;
- `scripting` — восстановление content script после обновления вкладки.

Доступ к страницам ограничен доменами `chatgpt.com` и `chat.openai.com`. Расширение не запрашивает `cookies`, `history`, `webRequest`, `debugger`, `nativeMessaging` или `<all_urls>`.

Подробнее: [PRIVACY.md](PRIVACY.md) и [SECURITY.md](SECURITY.md).

## Проверка разработки

Требуется Node.js 20 или новее:

```bash
npm run audit:extension
```

Команда проверяет синтаксис, автомат решений, сценарии service worker, восстановление вкладок, выбор последнего чата, отсутствие дублей, `at-most-once`, Manifest V3, минимальность разрешений и согласованность интерфейса с версией. GitHub Actions повторяет полный набор в пяти независимых циклах и создаёт ZIP с SHA-256.

## Статус

Текущая версия — **0.5.2 beta**. Она усиливает релизную надёжность версии 0.5.1: исправляет работу полного интерфейса, повторное открытие вкладок, отображение версии и сохранность несохранённых настроек при смене скина.

Предыдущая нативная версия для macOS на WebKit удалена из основной ветки проекта. `main` содержит только Chrome-расширение.
