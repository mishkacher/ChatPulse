# Восстановление зависших вкладок

ChatPulse 0.5.1 проверяет `discarded` и `frozen`, доступность content script, ошибки страницы и возраст последнего принудительного обновления. Неактивные вкладки без черновика и без текущей генерации обновляются каждые 5–15 минут. После reload расширение ждёт полной загрузки и гидратации ChatGPT, затем повторяет проверку перед отправкой команды.

Активная вкладка, обычная текущая генерация и пользовательский черновик не перезагружаются планово. Фоновая генерация восстанавливается только после зависания более 20 минут.
