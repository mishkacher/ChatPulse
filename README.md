# ChatPulse

**ChatPulse** — нативная утилита для строки меню macOS, которая следит за выбранными чатами ChatGPT и отправляет команду:

> продолжай и не останавливайся до технического лимита

Версия `0.2.0` использует собственный встроенный браузер на системном движке **WebKit**. Google Chrome больше не требуется.

## Как выглядит приложение

В строке меню macOS находится значок ChatPulse. По нажатию открывается небольшое меню:

- статус работы;
- одна команда **«Запустить / Остановить»**;
- ручная проверка;
- настройка интервала;
- открытие встроенного браузера;
- добавление текущего чата;
- список сохранённых чатов;
- журнал последних действий.

Отдельное окно **«Браузер ChatPulse»** похоже на упрощённый Safari: кнопки назад, вперёд, обновить, открыть ChatGPT, строка текущего адреса и кнопка **«Добавить чат»**.

## Что реализовано

- встроенный `WKWebView` на WebKit;
- постоянное локальное хранение cookies и данных входа;
- отдельное видимое окно для входа и выбора чатов;
- отдельное скрытое WebKit-окно для фоновых проверок;
- сохранение названий, URL и состояния чатов;
- интервалы 1, 2, 5, 10, 15 и 30 минут;
- собственный интервал от 30 секунд;
- защита от повторной отправки одной команды;
- ожидание завершения ответа;
- пассивная первая проверка после запуска;
- подтверждение появления отправленного сообщения;
- полностью русский пользовательский интерфейс и журнал;
- 20 автоматических проверок качества и модульные тесты.

## Важное изменение

Детектор технического лимита удалён. ChatPulse не пытается распознавать баннеры или тексты лимитов. Если поле ввода или кнопка отправки недоступны, попытка считается неуспешной и повторяется только при следующей плановой проверке.

## Главная логика

1. На первой проверке ChatPulse запоминает текущий ответ и ничего не отправляет.
2. При появлении нового ответа приложение снова только запоминает его.
3. Если на следующей проверке ответ не изменился, завершён и принадлежит ассистенту, отправляется команда продолжения.
4. Отпечаток обработанного ответа сохраняется на диске.
5. Один и тот же ответ не может получить команду дважды.

## Требования

- macOS 13 Ventura или новее;
- подключение к интернету;
- аккаунт ChatGPT;
- Swift 6 / Xcode Command Line Tools только для сборки из исходников.

Chrome, Playwright, Ollama и платные API не нужны.

## Установка из исходников

```bash
git clone https://github.com/mishkacher/ChatPulse.git
cd ChatPulse
git checkout agent/initial-chatpulse
bash scripts/install_app.sh
```

Приложение будет установлено в `/Applications/ChatPulse.app`.

## Первое использование

1. Нажмите значок ChatPulse в строке меню.
2. Выберите **«Открыть браузер ChatPulse…»**.
3. Войдите в ChatGPT внутри этого окна.
4. Откройте нужный чат.
5. Нажмите **«Добавить чат»** в окне браузера либо пункт добавления в меню ChatPulse.
6. Повторите для остальных чатов.
7. Выберите интервал и нажмите **«Запустить»**.

## Cookies и вход

Встроенный браузер использует постоянное хранилище WebKit. Вход сохраняется между запусками ChatPulse, но не связан с cookies, историей или вкладками приложения Safari.

### Ограничение входа через Google

Google может блокировать авторизацию внутри встроенных WebView. В таком случае используйте другой доступный способ входа в ChatGPT, например электронную почту, пароль, код, passkey или вход через Apple. Это ограничение политики Google, а не ошибка ChatPulse.

## Хранение настроек

```text
~/Library/Application Support/ChatPulse/settings.json
```

В настройках сохраняются:

- названия и URL чатов;
- включённое состояние;
- интервал проверки;
- отпечатки последних ответов;
- время последней успешной отправки.

Содержимое всей переписки в файл настроек не записывается.

## Ограничения alpha-версии

- интерфейс ChatGPT может измениться и потребовать обновления селекторов;
- встроенный вход должен быть проверен на реальном аккаунте;
- сборка подписывается локальной ad-hoc подписью, а не Apple Developer ID;
- приложение не обходит и не ускоряет сброс лимитов ChatGPT.
