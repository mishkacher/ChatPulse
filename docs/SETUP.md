# Установка и устранение проблем

## Требования для сборки

Установите Xcode Command Line Tools:

```bash
xcode-select --install
```

Проверьте Swift:

```bash
swift --version
```

## Чистая установка

Скопируйте весь блок целиком в Terminal:

```bash
cd ~
rm -rf "$HOME/ChatPulse-install"
git clone --depth 1 https://github.com/mishkacher/ChatPulse.git "$HOME/ChatPulse-install"
cd "$HOME/ChatPulse-install"
bash scripts/install_app.sh
```

Не выполняйте `git clone` внутри уже открытой папки ChatPulse: это создаёт вложенные каталоги `ChatPulse/ChatPulse` и затрудняет обновление.

## Сборка

```bash
bash scripts/build_app.sh
```

Результат:

```text
dist/ChatPulse.app
```

Во время сборки автоматически:

- компилируется release-версия Swift;
- из `Resources/ChatPulseIcon.svg` создаётся системная иконка `.icns`;
- формируется bundle `ChatPulse.app`;
- выполняется локальная ad-hoc подпись.

## Повторная установка из уже скачанной папки

```bash
cd "$HOME/ChatPulse-install"
git pull --ff-only
bash scripts/install_app.sh
```

Скрипт:

1. собирает свежую версию приложения;
2. останавливает уже запущенную копию ChatPulse;
3. устанавливает приложение в `/Applications`, если папка доступна для записи;
4. иначе использует `~/Applications` без запроса прав администратора;
5. запускает установленную копию.

Для ручного запуска:

```bash
open "/Applications/ChatPulse.app" 2>/dev/null || open "$HOME/Applications/ChatPulse.app"
```

## Первый вход

1. Откройте **«Браузер ChatPulse»** из строки меню.
2. Нажмите **«Войти ▾»**.
3. Выберите **«Войти по email / коду»** или **«Войти с passkey»**.
4. После входа откройте конкретный разговор.
5. Нажмите **«Добавить чат»**.
6. Выберите интервал проверки.
7. Нажмите **«Запустить»**.

Никакие разрешения Automation для Chrome или Safari не требуются.

## Вход по email / одноразовому коду

1. В браузере ChatPulse нажмите **«Войти ▾»**.
2. Выберите **«Войти по email / коду»**.
3. ChatPulse откроет официальный экран OpenAI и сфокусирует поле email.
4. Введите email и продолжите официальный процесс OpenAI.
5. Когда OpenAI пришлёт одноразовый код, введите его на странице входа.
6. Если на экране проверки доступна кнопка `Try with email`, используйте пункт **«Запросить код по email на этой странице»**.

ChatPulse не читает почту, не перехватывает код и не сохраняет email, пароль или одноразовые данные.

Если аккаунт был создан только через Google, OpenAI может не разрешить вход по email или сброс пароля. Это ограничение аккаунта, а не ошибка ChatPulse.

## Вход с passkey

1. Нажмите **«Войти ▾»**.
2. Выберите **«Войти с passkey»**.
3. ChatPulse проверит WebAuthn и найдёт официальный элемент Passkey / Ключ доступа.
4. Если OpenAI требует email перед показом passkey, введите email.
5. Подтвердите вход в системном окне macOS.

В зависимости от настройки подтверждение может использовать Touch ID, пароль Mac, iCloud Keychain, сторонний менеджер учётных данных или аппаратный ключ безопасности.

Passkey должен быть заранее добавлен в аккаунт OpenAI. Если ChatPulse сообщает, что ключ не найден:

1. войдите в ChatGPT через обычный Safari или официальное приложение;
2. откройте настройки безопасности аккаунта;
3. добавьте passkey;
4. повторите вход в ChatPulse.

## Огромное белое окно при входе

Версия `0.2.1` исправила обработку `window.open` и `about:blank`:

- основное окно больше не заменяется пустой страницей;
- дочерние окна ограничиваются видимой областью экрана;
- закрытие всплывающего окна корректно освобождает его WebKit-контроллер.

## Вход через Google

Google запрещает OAuth во встроенных WebView. ChatPulse распознаёт переход к Google OAuth, отменяет его и предлагает email-код или passkey вместо белого окна.

Сессия, созданная отдельно в Safari, не переносится в хранилище cookies встроенного `WKWebView`.

## Чат открыт, но команда не отправляется

Возможные причины:

- ответ обнаружен впервые и приложение ждёт следующую проверку;
- ChatGPT ещё генерирует;
- последнее сообщение принадлежит пользователю;
- этот ответ уже был продолжен;
- страница не загрузилась;
- поле ввода или кнопка отправки недоступны;
- чат был отключён или удалён во время проверки;
- интерфейс ChatGPT изменился.

Проверьте пункт **«Последние действия…»**.

## Сброс настроек

```bash
rm -rf "$HOME/Library/Application Support/ChatPulse"
```

## Удаление приложения

```bash
bash scripts/uninstall_app.sh
```

Скрипт удалит ChatPulse из `/Applications` и `~/Applications`, но оставит настройки пользователя.

Удаление данных входа WebKit выполняется через системное управление данными сайтов либо очистку профиля приложения.
