# Поддержка ChatPulse

## Перед созданием issue

1. Убедитесь, что используете последнюю версию из GitHub Releases.
2. Перезапустите ChatPulse и повторите действие.
3. Проверьте [инструкцию установки](docs/SETUP.md) и [известные ограничения](README.md#известные-ограничения).
4. Для проблем сборки выполните:

```bash
make preflight
```

## Открытые сообщения об ошибках

Используйте шаблон **Сообщение об ошибке** в GitHub Issues.

Укажите:

- версию ChatPulse;
- версию macOS и модель Mac;
- способ установки;
- безопасные шаги воспроизведения;
- ожидаемое и фактическое поведение.

Не публикуйте:

- cookies;
- приватные URL разговоров;
- email и одноразовые коды;
- passkey или сведения менеджера паролей;
- содержимое личной переписки.

## Уязвимости

Для проблем безопасности используйте приватный GitHub Security Advisory. Правила описаны в [SECURITY.md](SECURITY.md).

## Границы поддержки

ChatPulse взаимодействует с веб-интерфейсом ChatGPT. После изменений сайта может потребоваться новая версия селекторов. Наличие аккаунта, доступность конкретного способа входа, лимиты сервиса и работа внешней инфраструктуры не контролируются приложением.
