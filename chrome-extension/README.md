# ChatPulse 0.5.0 beta — расширение Google Chrome

Эта версия работает внутри обычного Google Chrome и использует уже авторизованный профиль браузера. Отдельный вход, WebKit, Safari Extension, сертификат Apple Developer и подпись приложения не требуются.

## Установка без подписи

1. Скачайте ZIP beta-сборки из GitHub Actions либо скачайте репозиторий.
2. Распакуйте архив в постоянную папку, которую не будете удалять или перемещать.
3. Откройте в Chrome:

```text
chrome://extensions
```

4. Включите **Режим разработчика** в правом верхнем углу.
5. Нажмите **Загрузить распакованное расширение**.
6. Выберите папку `chrome-extension` — внутри неё должен находиться `manifest.json`.
7. Закрепите ChatPulse на панели Chrome через меню расширений.

Chrome может показывать предупреждение о расширениях в режиме разработчика. Для локальной личной установки это ожидаемо.

## Первый запуск

1. В обычной вкладке Chrome войдите в ChatGPT через Google или другой доступный способ.
2. Откройте конкретный разговор с URL вида `https://chatgpt.com/c/...`.
3. Нажмите значок ChatPulse.
4. Нажмите **Добавить текущий чат**.
5. Добавьте остальные разговоры.
6. Выберите интервал.
7. Нажмите **Запустить**.

После запуска первая проверка каждой новой сессии только фиксирует текущее состояние. Новый ответ также сначала фиксируется. Команда отправляется лишь на следующей проверке, если ответ завершён, принадлежит ассистенту и не изменился.

## Полный интерфейс

В popup нажмите **Открыть полный интерфейс**. Там доступны:

- точный текст команды;
- интервал от 30 секунд;
- список и состояние чатов;
- включение и отключение отдельных разговоров;
- ручная проверка;
- журнал действий;
- скины **macOS** и **ChatPulse Preview**.

## Разрешения

Расширение запрашивает только:

- `storage` — локальные настройки и журнал;
- `tabs` — поиск и открытие выбранных разговоров;
- `alarms` — плановые проверки;
- `scripting` — восстановление content script после обновления вкладки;
- доступ только к `https://chatgpt.com/*` и `https://chat.openai.com/*`.

Расширение не запрашивает cookies, историю браузера, пароли, почту, `debugger`, `webRequest`, `nativeMessaging` или доступ ко всем сайтам.

## Важные ограничения beta

- Chrome должен быть запущен.
- В используемом профиле Chrome должен быть выполнен вход в ChatGPT.
- Интерфейс ChatGPT может измениться и потребовать обновления DOM-селекторов.
- Chrome может откладывать фоновые alarms для экономии энергии.
- Режим разработчика предназначен для локальной установки. Для массового распространения потребуется Chrome Web Store или управляемая корпоративная политика.
- До ручного теста на реальном аккаунте версия остаётся beta и не заменяет публичный релиз `0.4.0` в `main`.

## Проверка разработки

Из корня репозитория:

```bash
npm run audit:extension
```

Проверяются синтаксис JavaScript, автомат решений, at-most-once семантика, нормализация URL, сохранение настроек, Manifest V3 и минимальность разрешений.
