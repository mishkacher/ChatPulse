# Поддержка ChatPulse

## Расширение не загружается

1. Распакуйте архив полностью.
2. Откройте `chrome://extensions`.
3. Включите **Режим разработчика**.
4. Нажмите **Загрузить распакованное расширение**.
5. Выберите именно папку `chrome-extension`, содержащую `manifest.json`.

Не удаляйте и не перемещайте выбранную папку после установки.

## Не добавляется чат

Откройте конкретный разговор с URL вида `https://chatgpt.com/c/...`. Главная страница ChatGPT, временный чат без постоянного URL или сторонний домен не добавляются.

## Показывается «не выполнен вход»

Войдите в ChatGPT в том же профиле Google Chrome. ChatPulse не выполняет отдельную авторизацию и использует существующую сессию вкладки.

## Команда не отправляется

- дождитесь завершения ответа;
- первая проверка новой сессии всегда пассивная;
- после нового ответа пропускается один полный интервал;
- проверьте, что чат включён в списке;
- откройте страницу настроек и посмотрите журнал;
- нажмите **Проверить сейчас** для диагностики.

## Команда отправилась один раз и больше не повторяется

Это ожидаемая защита `at-most-once`. Один и тот же отпечаток ответа не получает команду повторно, даже если DOM не подтвердил отправку после фактического клика.

## После обновления ChatGPT появилась ошибка

Интерфейс ChatGPT может измениться. Создайте issue и приложите безопасный фрагмент журнала без URL, содержимого переписки и данных аккаунта.
