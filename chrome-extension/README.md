# ChatPulse 0.5.2 beta — расширение Google Chrome

Эта версия работает внутри обычного Google Chrome и использует уже авторизованный профиль браузера. Отдельный вход, WebKit, Safari Extension, сертификат Apple Developer и подпись приложения не требуются.

## Установка без подписи

1. Скачайте ZIP beta-сборки из GitHub Actions либо скачайте репозиторий.
2. Распакуйте архив в постоянную папку.
3. Откройте в Chrome `chrome://extensions`.
4. Включите **Режим разработчика**.
5. Нажмите **Загрузить распакованное расширение**.
6. Выберите папку `chrome-extension`.
7. Закрепите ChatPulse на панели Chrome.

## Первый запуск

1. Войдите в ChatGPT в обычной вкладке Chrome.
2. Откройте разговор с URL вида `https://chatgpt.com/c/...`.
3. Нажмите значок ChatPulse.
4. Нажмите **Добавить текущий чат**.
5. Выберите интервал и нажмите **Запустить**.

Первая проверка фиксирует baseline. Команда отправляется только после следующей неизменной проверки завершённого ответа ассистента.

## Исправления 0.5.2

- popup и полный интерфейс получают версию из Manifest;
- страница настроек добавляет последний использованный конкретный чат ChatGPT;
- кнопка «Открыть» активирует существующую вкладку без дубля;
- несохранённая команда и интервал не теряются при смене скина;
- журнал и карточки показывают последнее восстановление вкладки;
- service-worker integration test проверяет восстановление и `at-most-once`.

## Восстановление зависших чатов

ChatPulse обнаруживает выгруженные или замороженные вкладки, восстанавливает content script, синхронизирует безопасные неактивные вкладки и не перезагружает активную вкладку, обычную генерацию или вкладку с пользовательским черновиком.

## Разрешения

Расширение использует только `storage`, `tabs`, `alarms`, `scripting` и доступ к `chatgpt.com`/`chat.openai.com`. Оно не запрашивает cookies, историю, пароли, почту, `debugger`, `webRequest`, `nativeMessaging` или доступ ко всем сайтам.

## Обновление установленной копии

После замены файлов откройте `chrome://extensions` и нажмите **Обновить** на карточке ChatPulse. Затем один раз перезагрузите уже открытые вкладки ChatGPT.

## Проверка разработки

```bash
npm run audit:extension
```

Проверяются синтаксис, модель решений, service worker, восстановление вкладок, выбор чата, отсутствие дублей, `at-most-once`, Manifest V3 и минимальность разрешений.
